## Run Locally

Clone the project

```bash
  git clone https://github.com/LeandroVignoni/intro-rxjs-vanilla-js.git
```

Go to the project directory

```bash
  cd intro-rxjs-vanilla-js
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run start
```
